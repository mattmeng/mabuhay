A Ruby GUI framework designed to leverage web technologies without the need to write anything but Ruby.

## Installation

Add this line to your application's Gemfile:

```bash
gem install mabuhay
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install mabuhay
```
