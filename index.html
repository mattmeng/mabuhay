    Mabuhay

    A Ruby GUI framework designed to leverage web technologies without the need to write anything but Ruby.

    Installation

    Add this line to your application's Gemfile:

    gem 'mabuhay'

    And then execute:

    $ bundle

    Or install it yourself as:

    $ gem install mabuhay
